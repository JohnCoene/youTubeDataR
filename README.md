# youTubeDataR

Integrates R and the YouTube Data API.

### Functions ###

#### OAuth ####

* `youOAuth`

#### Find-family ####

* `findParams`
* `findParts`

#### Search ####

* `searchTube`

#### GET-family ####

* `getActivities`
* `getcaptions`
* `getChannels`
* `getChannelSections`
* `getComments`
* `getCommentThreads`
* `getCategories`
* `getLanguages`

### Install ###

`devtools::install_github("SocialFunction/youTubeDataR")`
